# Customer Support Agent 

A simple FAQ-based Customer Support Agent built in Python.  
The agent answers customer queries using predefined FAQs and ensures responses stay consistent with your data.  

---

## 📌 Features
- Loads predefined FAQs from `faq_data.py` (or a text file).  
- Loaded the FAQ text file (faq_data.txt) with load_instruction_from_file()
- Returns polite, FAQ-based responses.  
- Works both in command line and Google ADK.  
- Developed with assistance from ChatGPT
